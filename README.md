# SkillSphere — Online Learning Platform

A modern online learning platform where users can explore courses, watch lessons, and enroll in skill-based programs.

## 🔗 Live URL

[https://a8-skillsphere-app-mjfx.vercel.app](https://a8-skillsphere-app-mjfx.vercel.app)

## ✨ Key Features

- Browse 6+ premium courses across multiple categories
- User authentication with Email & Google (BetterAuth)
- Protected course details page (login required)
- Search and filter courses by category
- My Profile page with update functionality
- Fully responsive on mobile, tablet, and desktop
- Hero banner with Swiper.js animation
- Toast notifications
- 404 Not Found page
- Skeleton loader on data fetching

## 📦 NPM Packages Used

- `better-auth` — Authentication
- `swiper` — Hero slider animation
- `@heroui/react` — UI components
- `@heroui/toast` — Toast notifications
- `react-icons` — Icons
- `mongoose` — MongoDB
